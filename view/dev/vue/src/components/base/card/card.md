## 使用方法

| 参数            |      说明      |   类型 | 默认值 |
| --------------- | :------------: | -----: | -----: |
| width           |   card的宽度   | String |   100% |
| borderRadius    |  card的圆角值  | Number |      2 |
| backgroundColor | card的背景颜色 | String |   #fff |



```html
  <card>
    <div slot="content">
    </div>
  </card>
```
